# Proyecto Tecnologías Web

Este proyecto es una aplicación web escolar desarrollada como parte del curso de Tecnologías Web en la Benemérita Universidad Autónoma de Puebla. La aplicación está diseñada para gestionar procesos escolares de manera digital, utilizando tecnologías web modernas y control de versiones con Git.

# Tecnologías Utilizadas
# Frontend
HTML5
CSS3 (con Flexbox/Grid)
JavaScript 

# Backend
[Django/Node.js/Express/Laravel/etc.] 
Base de datos: [MySQL/PostgreSQL/MongoDB/SQLite]

Autenticación: JWT / Sessions
